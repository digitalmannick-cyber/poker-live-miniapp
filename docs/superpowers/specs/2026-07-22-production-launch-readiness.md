# 智牌屋正式上线发布前准备与验收清单

## 1. 当前结论

截至 2026-07-23，当前代码候选版本为 `3.39`，已上传微信开发版本，构建号为
`1.0.20260722.2325.v3.39`。代码自动化、干净包预览、三项受影响云函数部署和云端基础设施预检均通过，但**当前仍不建议立即提交正式审核**。

上线前仍有 4 类阻塞项：

1. 候选源码已经推送并创建标签，但回滚负责人尚未明确。
2. 广场文字评论、回复及公开昵称的服务端自动检测已部署，仍需在体验环境完成正常文字、违规文字和接口异常的真机验证；管理员事后删除继续作为人工兜底。本阶段不建设用户举报队列。
3. 好友邀请小程序码和 AI 提醒订阅消息当前均使用 `trial` 环境；正式发布后必须分别切换为 `release` 和 `formal`，否则普通用户可能仍被带到体验版，或提醒点击后无法进入正式版。
4. 微信公众平台只读实查已确认备案、体验版指向和客服能力；但 UGC 场景仍为“未声明”，隐私指引漏列公开昵称/头像等实际处理信息且仍声明未启用的麦克风用途，服务类目适配性也尚未形成提交证据。
5. 本机微信开发者工具提示版本过低；升级并重新完成一次干净预览前，模拟器结果不能作为发布证据。

满足本文 P0 条件后，才进入“提交审核”；审核通过后，还要完成发布日切换和真机回归，才能正式对外宣传。

## 5. 审核、发布和发布后顺序

### 阶段 A：提交审核

1. 完成全部 P0 项并重新生成体验版。
2. 用全新微信账号走一遍首次登录、创建场次、记录手牌、查看统计、添加好友和发布牌谱。
3. 填写服务类目、页面路径、功能说明和上述审核备注。
4. 提交审核；不要在审核期间修改体验版或云函数行为。

### 阶段 B：审核通过后的正式发布

1. 确认审核通过版本与 Git 发布标签、体验版构建号完全一致。
2. 发布小程序正式版。
3. 记录发布提交和版本号，执行 `tools/production-environment-switch.ps1 -Apply -ConfirmPublished -ExpectedCommit <发布提交> -ConfirmVersion <发布版本>`。脚本必须同时把 `SOCIAL_INVITE_QR_ENV_VERSION` 从 `trial` 切换为 `release`，把 `AI_REMINDER_MINIPROGRAM_STATE` 从 `trial` 切换为 `formal`；仅更新函数配置，保留其他全部环境变量，不轮换邀请密钥、不修改管理员名单。
4. 再运行一次不带参数的 `tools/production-environment-switch.ps1`，确认输出为 `invite=release; reminder=formal`。
5. 用一个从未加入体验名单的普通微信扫描新生成的小程序码，确认进入正式版邀请页；再触发一条 AI 提醒，确认点击进入正式版目标页面。
6. 验证微信分享卡片、搜索进入、冷启动和老用户升级。

### 阶段 C：上线后 24/72 小时

- 前 2 小时：每 15 分钟检查登录、云函数错误率、广场读取、评论和新增账号。
- 前 24 小时：每 2 小时检查一次，重点关注二维码、账号同步、缓存错乱、权限越权和 AI 超时。
- 24—72 小时：每天至少两次检查错误日志、管理员评论处置记录、云资源消耗和用户反馈。
- 出现跨账号数据、未匿名手牌、越权阅读或无法清除数据时，立即停止公开入口并回滚，不等待普通热修复。

## 6. 回滚标准

满足任一条件立即回滚或关闭对应能力：

- 不同微信账号看到彼此私有牌谱、Note、Leak、盈利或真实玩家名。
- 广场分享未完成 BB 化或非 Hero 玩家未匿名。
- 好友解除或范围变更后旧权限仍有效。
- 评论/昵称出现明显违规内容且无法及时处置。
- 云函数连续 5 分钟错误率超过 5%，或核心数据写入出现重复/丢失。
- 账号清除无法完成或清除后仍能恢复公开社交资料。

回滚顺序：先关闭广场/评论等可独立停用的高风险能力，再恢复上一稳定代码版本；数据库集合和索引不得自动删除。管理员配置异常时使用既有 fail-close 流程移除管理员能力。

## 7. 最终 Go / No-Go 门槛

只有同时满足以下条件才为 **GO**：

- P0 全部完成；
- `1105/1105` 或更多测试通过；
- 当前体验版对应唯一 Git 提交与发布标签；
- 三账号真机矩阵通过；
- 备案、类目、隐私指引和客服入口人工确认完成；
- 数据备份、告警、回滚负责人和发布后值守已经确定；
- 审核通过后已把邀请二维码切换到 `release`、AI 提醒切换到 `formal`，并由非体验成员验证。

否则结论为 **NO-GO**，继续保留体验版，不进行正式发布。
